Ongoing Personal Projects

These are some of the selected ongoing personal projects I've been working on.

Mobile Application for Effective Advertising

[Designer, Developer] Developed a complete system including Android mobile applications using Android Studio (iOS in progress), a web server, and a database (MySQL via phpMyAdmin). Benefits both the advertisers and customers by providing effective advertisements and benefits to customers, placing the system at a more competitive edge than other providers such as Facebook and YouTube. (Currently preparing for a start-up.)

4DOF/6DOF Motion Simulator

[Designer, Developer, Sales Manager] Using SolidWorks, designed and developed new hybrid 4DOF/6DOF motion simulators at an exceedingly low price compared to the market price. Used Thanos AMC controller to handle telemetry data from software and control actuators. Custom-ordered and imported self-designed parts of the simulators from China. (Currently preparing for a start-up.)

Navigation for the Blind

[Designer, Developer] A navigation system, including a mobile application, using RFID (radio-frequency identification) technology and a compass to provide more accurate directions to the blind. Can be easily updated as corresponding information for each RFID tag is stored in a database and the mobile app fetches data in real-time. (Will be working with the National Institute for the Blind, Visually Impaired and Deafblind (NIB) to employ the system in real life.)

American Sign Language to Text/Voice System

[Designer, Developer] A system allowing easier communication between signers and non-signers by detecting sign languages and converting them to text/voice. Google’s Soli system will be applied to enable more precise detection of hand gestures to differentiate the subtleties. (Will be working with the Deaf In Government (DIG) to employ the system in real life.)
